# CodeAlpha_Car_Price_Prediction

# Car Price Prediction Project

## Project Overview 
This project predicts car selling prices using Machine Learning with Python.  
The project uses Linear Regression to analyze and predict car prices based on different car features.

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## Dataset
Dataset Used:
- Car Data Dataset (.csv)

---

## Features Used
- Present Price
- Driven Kilometers
- Fuel Type
- Seller Type
- Transmission
- Owner
- Car Age

---

## Machine Learning Algorithm
- Linear Regression

---

## Features of Project
- Data Cleaning
- Feature Engineering
- Model Training
- Price Prediction
- Accuracy Checking
- Data Visualization

---

## Graphs Included
1. Actual Price vs Predicted Price
